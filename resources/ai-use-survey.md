## Purpose

The AI use survey helps the instructional team understand how students are using AI.

The goal is not to punish students for using AI. The goal is to encourage honest use and identify students who need help understanding their own code.

## Recommended Questions

Add these questions to the homework submission form or use them as a separate survey.

### Question 1

**Did you use AI on this assignment?**

Options:

- No
- Yes

### Question 2

**How did you use AI?**

Options:

- I did not use AI.
- I used AI to explain a concept.
- I used AI to explain an error message.
- I used AI to help write pseudocode.
- I used AI to review code I already wrote.
- I used AI to improve comments or README wording.
- I used AI to generate code.
- Other

### Question 3

**Briefly explain how you used AI.**

Use paragraph response.

### Question 4

**Can you explain all code in your submission?**

Options:

- Yes
- Mostly, but I have a few questions.
- No, I need help understanding part of my code.

### Question 5

**Do you want help reviewing your AI use with an IA or instructor?**

Options:

- No
- Yes

## IA Follow-Up

Flag the submission for the lead instructor if a student says:

- They used AI to generate code
- They cannot explain their code
- They need help understanding their submission

Do not accuse the student. Start with support.

Example:

```text
Thanks for being honest about your AI use. Let's review this together so you feel confident explaining your code.
```
