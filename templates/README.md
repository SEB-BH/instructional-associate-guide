## Student Intro Message

Send this message in each student's private Slack conversation by the end of Day 1.

```text
Hi @student! Welcome to the cohort.

This Slack conversation is your official communication channel with the Instructional Associates.

We will use this space for homework feedback, attendance follow-up, progress updates, and questions.

You can also reply here if you need help or if something affects your ability to complete work on time.
```

## Daily Attendance Post

Post this in the instructional channel every day at 10:00 AM.

```text
:spiral_calendar_pad: Attendance Update — [Day, Date]

Absent:
- students

Late:
- students
```

## Thursday Progress Recap

Post this in the instructional channel every Thursday morning.

```text
:bar_chart: Weekly Student Progress Recap — [Week #]

Attendance concerns:
- @student — [absences/tardies]

Students currently below 80% homework completion:
- @student — [percentage]

Students needing follow-up:
- @student — [reason]
```

## AI Use Follow-Up

```text
Hi @student, thanks for being honest about your AI use on this assignment.

Let's review the code together so you feel confident explaining how it works.

Please reply with the part you feel least confident about, and we can start there.
```
